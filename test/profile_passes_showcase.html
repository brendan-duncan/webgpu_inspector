Profile Passes — Timeline Showcase

This page renders a frame with five labeled GPU passes. Two of them are deliberately expensive; the other three are cheap. To see which is which:

  • Open WebGPU Inspector, switch to the Capture panel.
  • Check Profile Passes, then click Capture.
  • The timeline above the command tree shows one segment per pass, sized by GPU time.
  • Click a segment to jump to that pass in the command list.

The page itself does not request the timestamp-query feature and does not set timestampWrites on any pass — the inspector handles all of that when Profile Passes is enabled.
